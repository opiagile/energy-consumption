# Project energy-consumption 

## Prerequisites
 JDK 11
 
 Maven 3.8.1
 
 IDE your choice
 
## running the application

Run the app by mainclass in:
```shell script
br.com.landisgyr.energyconsumption.EnergyConsumptionApplication
```

Run the app using maven command:
```shell script
mvn clean spring-boot:run
```
